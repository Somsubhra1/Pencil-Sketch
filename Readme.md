# Pencil Sketch

A Pencil Sketch image modifier using OpenCV and SciPy.

## File Descriptions

- [Main.ipynb](Main.ipynb) - This file uses OpenCV to implement Pencil Sketch.
- [Main2.ipynb](Main2.ipynb) - This file uses SciPy library to implement the same.
